## Pages For Frontend

### **Home Page**

-   **Description**: Shows all the todos available.
-   **Functionality :**
    -   ` Go to details of One todo.`
    -   ` Create new Todo.`

### **Todo Page**

-   **Functionality :**

    -   ` Update the current todo.`
